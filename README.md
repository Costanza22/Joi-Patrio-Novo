

# 🌟 **Joi Patrio** 🌟  
**📜 Documentando e Preservando o Patrimônio Histórico de Joinville**  

---

## 🏛️ **Tema do Projeto**  
O **Joi Patrio** é uma plataforma web dedicada à **documentação e valorização do patrimônio histórico de Joinville**, com foco nos **casarões históricos**. Com uma interface moderna e interativa, o projeto conecta cidadãos e visitantes à rica herança cultural da cidade.

## 📝 **Resumo do Projeto**  
O projeto **Joi Patrio** nasceu para **preservar** e **promover** a história de Joinville. Ele cataloga casarões históricos com:  
- **Detalhes arquitetônicos.**  
- **Histórias fascinantes.**  
- **Mapas interativos.**  

💡 **Inspiração**: Valorizar a memória local e tornar o patrimônio acessível para todos.

---

## 🎯 **Objetivos**  
1. 📌 Desenvolver uma aplicação para catalogar casarões históricos.  
2. 🗺️ Oferecer um mapa interativo para explorar esses casarões.  
3. 👥 Engajar a comunidade na preservação cultural.  

---

## ⚙️ **Especificações Técnicas**  

### 🔑 **Funcionalidades Principais**  
| ✅ **Funcionalidade**                                     | ✅ **Descrição**                                                                 |
|---------------------------------------------------------|---------------------------------------------------------------------------------|
| 📷 **Registro de Casarões**                              | Cadastrar informações detalhadas com imagens.                                   |
| 🌟 **Favoritos**                                         | Marcar casarões preferidos para acesso rápido.                                  |
| 📝 **Sugestões da Comunidade**                           | Enviar novos casarões para análise e inclusão.                                  |
| 🔍 **Busca Inteligente**                                 | Encontrar casarões por nome ou localização.                                     |
| 🗺️ **Mapa Interativo**                                   | Localizar casarões diretamente no mapa virtual.                                |

### 🛠️ **Requisitos Não-Funcionais**  
- **Responsividade** 📱: Interface adaptável para dispositivos móveis.  
- **Acessibilidade** ♿: Inclusão de usuários com deficiência.  
- **Segurança** 🔒: Proteção de dados dos usuários.

## 📝 **Limitações e Expansão**  
- **Limitação Atual**: O projeto **Joi Patrio** está focado na cidade de **Joinville**, com informações detalhadas sobre os casarões históricos dessa localidade.  
- **Implementação Futura**: Há planos de expandir o projeto para outras cidades, com a colaboração das prefeituras locais, para registrar e valorizar o patrimônio histórico de outras regiões.  
- **Acordo com a Prefeitura**: O projeto tem o objetivo de trabalhar em parceria com a Prefeitura de Joinville e outras instituições públicas, visando o incentivo à preservação do patrimônio cultural da cidade.  

---

## 🎨 **Design e Estilo**  
- Fundo inspirado na **arquitetura histórica de Joinville**.  
- Ícones de navegação modernos (🔍, 📜, 🗺️).  
- Paleta de cores: tons de **marrom** e **dourado**, remetendo à história.  

---

## 🌍 **Tour Virtual entre os Casarões**  
Em uma **futura implementação**, o projeto oferecerá uma funcionalidade de **tour virtual**, permitindo que os usuários explorem os casarões históricos de Joinville de maneira imersiva e interativa, diretamente da plataforma.
